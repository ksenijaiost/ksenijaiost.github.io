Как добавить индивидуальный дизайн других игроков в автомате Ателье сестёр Эйбл?

Как только вы откроете ателье сестёр Эйбл, вам будет доступен автомат в конце помещения, воспользовавшись которым, вы сможете добавлять паттерны* других игроков.

*Паттерны (от англ. patterns — узоры).

Искать паттерны в автомате можно следующими способами:

  • Код автора MA — ......... По этому коду можно посмотреть все рисунки автора и сохранить их себе.
  • Код рисунка MO — ........ По этому коду вы найдёте один рисунок, который можно сохранить себе.
  • Поиск по критериям: название индивидуального дизайна и/или его тип.

Вы можете добавить любой дизайн, но, к сожалению, у вас есть ограничение по слотам. Дополнительные слоты можно получить, купив расширение в мэрии, в банкомате Нука.

Если ищете простой способ найти нужный дизайн, то посетите наш канал, либо воспользуйтесь Pinterest.

Создайте свой собственный дизайн

Вы можете использовать приложение на Android/iOS «Nintendo Switch Online» для импорта старых пользовательских дизайнов из Animal Crossing: New Leaf, Animal Crossing: Happy Home Designer и из любого места в интернете — например, с сайтов вроде acpatterns.com, которые могут преобразовать изображение в загружаемый QR-код. Для этого понадобится приложение Nintendo Switch Online.

Как использовать приложение Nintendo (NookLink) для сканирования QR-кодов

  • Зайдите в приложение Nintendo Switch APP/Online (NookLink)
  • Нажмите на «Инд. рисунки».
  • Считайте QR-код при помощи камеры.
  • Зайдите в игру и откройте приложение «Дизайнер» на Нукофоне, нажмите + на консоли и добавьте паттерн на свободный слот.

Как добавлять паттерны через Нукофон в самой игре

Чтобы не ходить каждый раз в ателье за новыми паттернами, можно купить в мэрии режимы для рисунков — они позволяют пользоваться автоматом поиска паттернов прямо с Нукофона.
