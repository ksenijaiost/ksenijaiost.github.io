# acnh.github.io

Гайды чата ACNH (статический сайт для GitHub Pages).

## Как открыть сайт локально

В проекте нет сборки — это чистый HTML/CSS.

**Самый простой способ:**
1. Откройте файл `index.html` двойным кликом в браузере.

**Если нужен локальный сервер (рекомендуется для корректных относительных ссылок):**
1. Запустите любой статический сервер в корне репозитория.
2. Откройте `http://localhost:PORT`.

Примеры (любой из вариантов):
```bash
# Python
python -m http.server 8000

# Node
npx serve .
```

## Структура проекта

- `index.html` — главная страница со списком гайдов и ссылками.
- `terminology.html` — страница терминов (пример перенесённого гайда).
- `styles.css` — общий стиль для всех страниц.

## Как добавлять новые страницы гайдов

1. Создайте новый файл `*.html` в корне (например, `rules.html`).
2. Скопируйте шапку/подвал из существующих страниц, чтобы сохранять единый стиль.
3. Подключите `styles.css` и проверьте, что есть ссылки:
   - на главную (`index.html`);
   - на чат и канал Telegram:
     - https://t.me/acnhchatru
     - https://t.me/acnhtime
4. Добавьте ссылку на новую страницу в `index.html`.

## Примечания по коду

- Сайт полностью статический, без JS и сборки.
- Изображения подключаются по внешним ссылкам (Tilda CDN).
- Можно переносить контент из Tilda блоками и упрощать разметку.
